GLITCH LOOPER Effect Reference

← back to the app · how to use →

Every effect turns a still image into a seamless looping video, entirely in your browser — nothing is uploaded. Effects stack; the Envelope makes them breathe over the loop, and the Sequencer can switch each effect on and off across the loop's 8 steps. Glitch effects are grouped by how they process the image — binary (bytes / bits / file format) vs pixels. Tag: real genuinely corrupts encoded file bytes then re-decodes (unpredictable); sim is deterministic processing. 各エフェクトは静止画をシームレスなループ動画に変換します。処理はすべてブラウザ内で完結し、画像はアップロードされません。効果は重ねがけでき、Envelope でループ内の抑揚を、Sequencer でループを8ステップに分けて各エフェクトのオン/オフを切り替えられます。グリッチ系は処理方法で分類 — バイナリ（バイト/ビット/ファイル形式）か ピクセル か。タグ: real＝エンコード済みバイトを実際に破壊して再デコード（予測不能）、sim＝決定的な処理。
